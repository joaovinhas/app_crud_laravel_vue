# Laravel + Vue Authentication CRUD

## Description
Simple web application built with Laravel and Vue, featuring user authentication and a basic CRUD example.

## Features
- User authentication (Laravel Breeze)
- Create, read, update and delete records
- REST API
- Vue frontend
- Authentication-protected routes

## Tech Stack
- Laravel
- PHP
- Vue.js
- MySQL
- Vite

## Setup
1. Clone the repository
2. Copy `.env.example` to `.env`
3. Configure database credentials
4. Run migrations:
   ```bash
   php artisan migrate
